# LANDING_STRUCTURE — Структура лендинга

**Статус:** черновик, ожидает утверждения

## Предварительная структура блоков

### 1. Первый экран (Hero)
- Заголовок с главным оффером
- Подзаголовок с пояснением ценности
- CTA-кнопка «Пройти тест»
- Элемент доверия (кол-во прошедших, рейтинг)

### 2. Проблема
- Описание боли ЦА (маркетинг не работает, деньги сливаются)
- Узнаваемые ситуации собственника микробизнеса

### 3. Решение
- Что даёт тест: диагностика маркетинга
- Серые и красные зоны — визуальное объяснение
- Пошаговый план исправлений

### 4. Как это работает
- 3–4 шага: ответь на вопросы → получи отчёт → следуй плану → выручка растёт

### 5. Для кого этот тест
- Портрет ЦА: собственник малого бизнеса, сам занимается маркетингом или нанял подрядчика

### 6. Что вы получите
- Конкретный список результатов теста
- Примеры/скриншоты отчёта

### 7. Социальное доказательство
- Отзывы / кейсы / цифры

### 8. FAQ
- Частые вопросы и ответы

### 9. Финальный CTA
- Повторный призыв пройти тест
- Кнопка

---

*Структура будет уточняться после утверждения текстов.*
