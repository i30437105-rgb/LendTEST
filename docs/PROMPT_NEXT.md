## Кто ты и что делаешь

Ты — фронтенд-разработчик, продолжающий работу над лендингом **MetodZMS**. Проект находится в папке текущего рабочего каталога. Технология — чистый HTML/CSS/JS, без фреймворков.

## Продукт

Лендинг для продукта "краш-тест маркетинга". Пользователь отвечает на 22 вопроса да/нет и получает персональные рекомендации: какие элементы маркетинга работают, какие нет, что починить для роста выручки. ЦА — собственники бизнеса РФ с оборотом до 5 млн ₽/мес.

### Блок «Сканер инструментов» (v21) — отдельная страница scanner.html

Интерактивный блок-сканер 20 маркетинговых инструментов. Метафора: рентген-сканер проходит по сетке, обеляя квадраты — диагностика маркетинга.

**Файл:** `scanner.html` (полностью автономный, НЕ затрагивает index.html/styles.css/script.js)

**Что на экране:**
- Сетка 5×4 = 20 ячеек (96×96px, gap 12px, border-radius 18px)
- Каждая ячейка: два слоя — `cell-bg` (#1a1a2e тёмный) + `cell-white` (#ffffff, clip-path контролируемый)
- Сканер-линия: лаймовая (#c8f542) с glow-свечением (3 уровня box-shadow)
  - При загрузке: задержка 1 сек → спуск до 50% за 2.5 сек
  - Перетаскивается мышкой вверх-вниз в пределах сетки
  - При клике НЕ прыгает (фиксация текущей позиции через getBoundingClientRect перед отключением animation)
- Обеление: попиксельное через clip-path — белый слой раскрывается ровно до линии сканера
- Hover: scale(1.03) + box-shadow 0 4px 16px rgba(26,26,46,0.15)
- Первый ряд: 5 иконок Lucide (белые, 36px): globe (Рынок), chess-knight (Стратегия), megaphone (Маркетолог), binoculars (Конкуренты), coins (Юнит-экономика)

## Железные правила (краткая версия)

1. **Перед любой задачей** — прочитай PROJECT.md + DESIGN_STANDARDS.md
2. **После каждого изменения** — запиши в CHANGELOG.md
3. **НЕ создавай графику** — только Lucide, сохраняй в /assets/icons/
4. **НЕ начинай следующий экран** без утверждения предыдущего
5. **Десктопные правки НЕ распространяй** автоматически на мобайл
6. **Сначала точная копия референса** — потом замена текстов
7. **Иконки** подключать через `<img src="assets/icons/...">`, НЕ inline SVG
8. **Локальный сервер:** `npx serve -l 3000 .`
9. **После блока правок** — обновить ВСЕ документы, закоммитить, запушить
10. **Шрифт для кириллицы:** H1 = Unbounded, остальное = Manrope. Chakra Petch только для англ. текстов
