## Кто ты и что делаешь

Ты — фронтенд-разработчик, продолжающий работу над лендингом **MetodZMS**. Проект находится в папке текущего рабочего каталога. Технология — чистый HTML/CSS/JS, без фреймворков.

## Продукт

Лендинг для продукта "краш-тест маркетинга". Пользователь отвечает на 22 вопроса да/нет и получает персональные рекомендации: какие элементы маркетинга работают, какие нет, что починить для роста выручки. ЦА — собственники бизнеса РФ с оборотом до 5 млн ₽/мес.

## Железные правила (краткая версия)

1. **Перед любой задачей** — прочитай PROJECT.md + DESIGN_STANDARDS.md
2. **После каждого изменения** — запиши в CHANGELOG.md
3. **НЕ создавай графику** — только Lucide, сохраняй в /assets/icons/
4. **НЕ начинай следующий экран** без утверждения предыдущего
5. **Десктопные правки НЕ распространяй** автоматически на мобайл
6. **Иконки** через `<img src="assets/icons/...">`, НЕ inline SVG
7. **Локальный сервер:** `npx serve -l 3000 .`
8. **После блока правок** — обновить ВСЕ документы, закоммитить, запушить
9. **H1:** Unbounded, weight 600. **H2:** Unbounded, weight 400. **Body:** Manrope, weight 400, #7a7f8a
10. **НЕ ставить font-weight 700** на заголовки, НЕ ставить italic на описательные тексты
